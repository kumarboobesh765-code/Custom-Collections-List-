## üöÄ Installation Steps (Shopify)

Follow these steps carefully üëá

---

### **Step 1: Open Shopify Admin**

* Go to **Shopify Admin ‚Üí Online Store**

---

### **Step 2: Edit Theme Code**

* Click the **three dots (‚ãÆ)** next to your active theme
* Select **Edit code**

---

### **Step 3: Create CSS File**

* Go to **Assets**
* Click **Add a new asset**
* Select **CSS**
* Name it exactly:

```
brand-collection-slider.css
```

* Paste the CSS code into this file
* Click **Save**

---

### **Step 4: Create JavaScript File**

* Go to **Assets**
* Click **Add a new asset**
* Select **JavaScript**
* Name it exactly:

```
brand-collection-slider.js
```

* Paste the JavaScript code into this file
* Click **Save**

---

### **Step 5: Create Section File**

* Go to **Sections**
* Click **Add a new section**
* Name the file exactly:

```
brand-collection-slider.liquid
```

* Paste the provided Liquid code into this file
* Click **Save**

---

### **Step 6: Add Section to Theme**

* Go to **Online Store ‚Üí Themes**
* Click **Customize**
* Navigate to the page where you want the slider (Home / Collection / Product)
* Click **Add section**
* Select **Brand Collection Slider**
* Click **Save**

## üß© How It Works

* Each block represents **one Shopify collection**
* Clicking a brand logo redirects to its collection page
* If no custom logo is added, the collection image is used automatically
